Quick Start Guide

1. Go to Settings → Connections and enter your Blogger Blog ID and Google OAuth Client ID.

2. Click Connect Google to authenticate your Blogger account.

3. Optional: Add your Claude API key in Settings → AI Engine for AI-powered articles.

4. Go to Live Events and click Fetch All to load today's sports events.

5. Click Compose on any event or go to Compose to write and post articles.

6. Enable the Scheduler for fully automatic posting.

FAQ

Is my API key safe? Keys are stored only in your browser's localStorage and never sent to any server except the respective APIs (Anthropic, Blogger, imgbb).

Why does my scheduler stop? The scheduler runs while the tab is open. For always-on scheduling, keep the tab active or use a scheduled task on a server.

How do I make AdSense-safe posts? Enable word count gate (400+ words), content scanner, and nofollow links in Settings → AdSense.

Can I use this without Claude AI? Yes — the fallback template system generates structured HTML articles automatically without any API key.
